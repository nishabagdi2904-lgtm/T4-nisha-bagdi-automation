# T4-nisha-bagdi-automation
Pravaah Consulting Automation Script

End-to-End Procedure (Login to Logout)

1. Login to Application
Step 1.1: Launch the application URL in browser.
Step 1.2: Enter valid Username.
Step 1.3: Enter valid Password.
Step 1.4: Click on Login button.
Verification:
User should be redirected to the Dashboard/Home page.
Username should be visible on the top.
No error message should appear.

2. Add New Generator
Step 2.1: Navigate to Generator section.
Step 2.2: Click on Add New Generator button.
Enter Generator Details
Step 2.3: Enter Generator Name
Example: Nisha N.B
Step 2.4: Enter Internal Account Number
Example: 321963741852
Verification:
Fields should accept valid input.
No validation error should appear.

4. Fill Billing Information
Navigate to the Billing Information section and fill the following:
Step 3.1: Select Industry (from dropdown).
Step 3.2: Enter Street Address.
Step 3.3: Enter valid Email ID.
Step 3.4: Enter valid Phone Number.
Verification:
Email should follow proper format.
Phone should accept only numbers.
Required field validations should work
.
4. Generate and Verify Latitude/Longitude
Step 4.1: Click on Generate Latitude/Longitude button.
Step 4.2: System auto-populates the latitude and longitude fields.
Verification:
Both fields should not be empty.
Values should be in numeric format.
Coordinates should match the entered address.

5. Copy Billing Information
Step 5.1: Click on Copy Billing Information checkbox/button.
Verification:
Shipping/Service address should auto-fill with billing details.
All copied fields should match exactly.

6. Fill Service Hours
Navigate to Service Hours section.
Step 6.1: Enter Open Time.
Step 6.2: Enter Lunch Time.
Step 6.3: Enter Close Time.
Step 6.4: Click on Copy hours to all days.
Verification:
All days should reflect same service hours.
Time format should be correct.

7. Route Assignment
Navigate to Route Assignment section.
Add Service
Step 7.1: Click on Add Service.
Fill the following:
Route Name = SanjuktaRoute
Service Type = Medical Waste
Frequency = Multiple Times Weekly
Select Weekdays = Thu / Fri / Sat / Sun
Scope of Work = Select 2 Containers
Select Anchor Date (dynamic current date)
Increase Quantity for both containers
Step 7.2: Click on Add Service to Route
Verification:
Service should appear in Route list.
Selected weekdays should be saved.
Quantity should update correctly.

8. Price Book
Navigate to Price Book section.
Step 8.1: Toggle View Mode.
Step 8.2: Update service prices for both containers.
Verification:
Updated price should be saved.
Price should reflect correctly in summary.

9. Save Generator
Step 9.1: Click on Save/Submit button.
Verification:
Success message should appear.
Generator should be visible in Generator List.

10. Logout
Step 10.1: Click on Profile/User icon.
Step 10.2: Click on Logout.
Verification:
User should be redirected to Login page.
Back button should not reopen session.
